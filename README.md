🎬 YouTube Clone

This is a simple YouTube Clone project built using HTML and CSS. It replicates the basic layout of YouTube including the header, sidebar, and video grid.

⸻

🚀 Live Demo

👉 https://mageshvarr106-dot.github.io/Youtube-clone/

⸻

📌 Features
	•	YouTube-style homepage layout
	•	Video thumbnails with titles and channel info
	•	Sidebar navigation menu
	•	Header with search bar
	•	Responsive design (basic mobile support)

⸻

🛠️ Technologies Used
	•	HTML
	•	CSS
